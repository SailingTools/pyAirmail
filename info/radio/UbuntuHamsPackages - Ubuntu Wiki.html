Attention users of these packages

There is a team established here for hams using Ubuntu Linux. This page has more links to the wiki, and to the associated mailing list.

Attention developers and debian Packagers

There is a team established here for ham developers and packagers. This team is subscribed to bugs for all of the Ubuntu packages listed here. The bug volume is low, so there's not a lot of mail. If you are able to help with bugs for these packages, please join the development team.
